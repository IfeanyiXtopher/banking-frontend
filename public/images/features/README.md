## How to download

1. Open the link in your browser.
2. Click **Download free** (Unsplash) or **Free download** (Pexels).
3. Rename the file to match the table (e.g. `global-transfers.jpg`).
4. Place it in `banking-frontend/public/images/features/`.
5. Refresh the landing page — images load automatically.

Until files are added, the UI shows an animated gradient placeholder with an icon.
