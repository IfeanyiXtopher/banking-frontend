# Trusted by — partner marks

Place **25** logo images in this folder. The landing page marquee loads them from `src/data/trustedByLogos.ts`.

Current filenames (see that file to change):

- `Client_List.jpg`
- `Client_List (1).jpg` … `Client_List (24).jpg`

Public URL prefix: `/images/trusted_by/<filename>` (spaces and parentheses are URL-encoded automatically).
